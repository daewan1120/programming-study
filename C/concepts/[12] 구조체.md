# Chapter 12 - 구조체 (Structure)

## 구조체란?
서로 다른 크기의 데이터들을 메모리 상에 연속적으로 배치하여 하나의 새로운 단위(Entity)를 만드는 과정

ex) 만약 학생 100명의 학번(int), 학점(float), 이름(char[20])을 관리할 때

- 구조체 미사용: int ids[100];, float grades[100];, char names[100][20]; 세 개의 배열을 따로 관리해야 하며, 인덱스가 어긋나면 데이터가 꼬입니다.

- 구조체 사용: Student students[100]; 하나의 배열로 모든 속성을 응집력 있게 관리합니다.


---

## 구조체 정의 및 선언
```C
#include <stdio.h>

// 1. 구조체 정의 (설계도)
struct Student {
    int id;          // 학번
    char name[20];   // 이름
    float grade;     // 학점
};

int main() {
    // 2. 구조체 변수 선언 및 초기화
    struct Student s1 = {202401, "홍길동", 4.5};

    // 3. 멤버 접근 (마침표 연산자 사용)
    printf("학번: %d\n", s1.id);
    printf("이름: %s\n", s1.name);
    printf("학점: %.1f\n", s1.grade);

    return 0;
}
```

## 구조체 포인터
구조체 변수를 포인터로 가리킬 때는 마침표(.) 대신 화살표 연산자(->)를 사용


```c
struct Student s1 = {1, "Kim", 3.8};
struct Student *ptr = &s1;

printf("%d\n", (*ptr).id); 
printf("%d\n", ptr->id);   // 두 표현은 동일하나 이 방식을 많이 사용
```

## 구조체 포인터
구조체 변수를 포인터로 가리킬 때는 마침표(.) 대신 화살표 연산자(->)를 사용


```c
// 패턴: typedef struct [구조체이름] { ... } [별칭];
typedef struct Node {
    int data;
    struct Node* next;
} Node; // 'struct Node' 대신 'Node'만 사용 가능

Node n1;
```

## 함수에 구조체 전달하기
구조체 변수를 포인터로 가리킬 때는 마침표(.) 대신 화살표 연산자(->)를 사용


```c
typedef struct {
    char name[20];
    int level;
} Player;

// 나쁜 예: 구조체 전체를 복사해서 넘김 (메모리 낭비)
void updateLevel_Bad(Player p) { 
    p.level++; // 원본은 변하지 않음
}

// 좋은 예: 주소만 넘김 (효율적이고 원본 수정 가능)
void updateLevel_Good(Player* p) {
    p->level++; // 원본의 레벨이 올라감
}

int main() {
    Player p1 = {"Hero", 1};
    updateLevel_Good(&p1); // 주소를 전달
}
```
