# Chapter 13 - 동적 할당(Dynamic Allocation)

## 메모리의 두 영역
- 스택(Stack): 우리가 지금까지 쓴 일반 변수들이 사는 곳. 함수가 끝나면 자동으로 사라짐. (고정 크기)
- 힙(Heap): malloc으로 빌려온 메모리가 사는 곳. 우리가 직접 반납하기 전까지 영원히 존재함. (가변 크기)

---

## 동적 할당의 3단계 프로세스

#### ① 빌리기: malloc()

malloc은 메모리(Memory)를 할당(Allocation)하는 역할

```c
// 패턴: 포인터 = (타입*)malloc(sizeof(타입));
int* ptr = (int*)malloc(sizeof(int));
```

- malloc은 주소만 주기 때문에, 그 주소가 int인지 char인지 알려줘야 함
- 메모리가 부족하면 NULL을 반환

#### ② 쓰기: 포인터처럼 사용

할당받은 ptr은 일반 배열이나 변수처럼 똑같이 사용하면 됨

```c
*ptr = 100;
printf("%d", *ptr);
```

#### ③ 갚기: free()

힙 영역은 자동으로 청소되지 않음!
빌린 메모리를 안 갚으면 '메모리 누수(Memory Leak)'가 발생

```c
free(ptr); // 사용 끝난 즉시 반납
```

## 구조체와 동적 할당의 결합

```c
typedef struct Node {
    int data;
    struct Node* next;
} Node;

int main() {
    // 1. 노드 크기만큼 힙에 메모리 할당
    Node* newNode = (Node*)malloc(sizeof(Node));

    // 2. 값 채우기
    newNode->data = 10;
    newNode->next = NULL;

    // 3. 사용 후 해제
    free(newNode);
}
```
