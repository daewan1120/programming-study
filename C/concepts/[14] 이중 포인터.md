# Chapter 14 - 이중 포인터 (Double Pointer)

## 이중 포인터란?
포인터 변수의 주소값을 저장하는 변수 (포인터를 가리키는 포인터)

구조: 데이터 → 포인터(*) → 이중 포인터(**)
- int pp는 int *p가 위치한 메모리 주소를 담고 있으며, *pp를 하면 p의 값(주소)에 접근하고, pp를 하면 실제 int 데이터에 접근함

---

## 이중 포인터 사용 이유
함수 내부에서 함수 외부의 포인터 값을 변경하기 위해 사용

```c
#include <stdio.h>

void changeAddress(int **pp) {
    static int newValue = 200; 
    *pp = &newValue; // 포인터 p가 가리키는 주소를 newValue의 주소로 변경
}

int main() {
    int n = 100;
    int *p = &n;

    printf("변경 전: %d\n", *p); // 100
    
    changeAddress(&p); // 포인터 p의 주소를 전달
    
    printf("변경 후: %d\n", *p); // 200
    return 0;
}
```

## 자료구조에서의 이중 포인터 활용 (Linked List)
- Head 노드를 관리할 때 필수적
- 함수 안에서 리스트의 시작 주소를 바꿔야 할 때 이중 포인터를 사용

```c
typedef struct Node {
    int data;
    struct Node* next;
} Node;

// 이중 포인터를 사용하여 메인 함수에 있는 head의 값을 직접 변경
void initHead(Node** headPtr) {
    *headPtr = (Node*)malloc(sizeof(Node));
    (*headPtr)->data = 1;
    (*headPtr)->next = NULL;
}

int main() {
    Node* head = NULL;
    
    // head 포인터의 주소를 넘겨서 함수 안에서 할당받게 함
    initHead(&head); 
    
    printf("Head Data: %d\n", head->data);
    free(head);
    return 0;
}
```

---
## 핵심 연산자 구분

- `pp`: 이중 포인터 변수 자체의 값 (가리키는 포인터 변수의 주소)

- `*pp`: 이중 포인터가 가리키는 포인터 변수의 값 (실제 데이터의 주소)

- `**pp`: 최종적인 실제 데이터 값
