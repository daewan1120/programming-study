# 1017. 포인터 배열을 이용한 문자열 정렬

**설명**  
두 개의 문자열을 입력받아 각각 배열 `s1`, `s2`에 저장합니다.
두 문자열 중 사전 순으로 더 앞에 오는 문자열을 가리키는 포인터 변수 `ptr`을 선언하여, 해당 포인터를 통해 문자열을 출력하세요.

###### **입력**  
첫 줄과 둘째 줄에 각각 하나의 단어가 입력된다. (각 최대 100자)

###### **출력**  
사전 순으로 더 앞선 단어를 출력한다.

###### 입력
```
apple
banana
```

###### 출력
```
apple
```
