# 1016. 포인터를 이용한 합계 누적

**설명**  
정수 N과 N개의 정수를 입력받습니다. 그 후, 포인터를 인자로 받는 함수 `void getSum(int *arr, int n, int *result)`를 작성하여 배열의 모든 요소의 합을 계산하세요.

※ 이 함수는 값을 반환하지 않고, 세 번째 인자로 받은 포인터 `result`가 가리키는 main 함수의 `sum` 변수에 결과값을 직접 저장해야 합니다.

###### 입력 1
```
5
1 2 3 4 5
```

###### 출력 1
```
15
```

###### 입력 2
```
3
10 20 30
```

###### 출력 2
```
60
```
