# 1020. 문자열 배열 사전순 정렬

## **설명**  
메인 함수에 여러 개의 문자열(포인터 변수)을 담고 있는 포인터 배열이 선언되어 있습니다. 이 배열의 주소를 받아 내부 문자열들을 사전순(오름차순)으로 정렬하는 함수를 작성하세요.

###### **입력**  
첫째 줄에 정렬할 문자열의 개수 $N$ ($1 \le N \le 10$)이 주어진다.

둘째 줄부터 $N$개의 줄에 걸쳐 공백이 없는 영어 소문자 문자열이 하나씩 주어진다.

(각 문자열의 길이는 1자 이상 20자 이하)

###### **출력**  
사전순으로 정렬된 문자열을 첫째 줄부터 $N$개의 줄에 걸쳐 하나씩 출력한다.

###### **입력 예시 1**
    3
    banana
    apple
    cherry

###### **출력 예시 1**
    apple
    banana
    cherry
