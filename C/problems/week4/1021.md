# 1021. 재귀를 이용한 이진수 출력

## **설명**  
10진수 양의 정수 $N$이 주어졌을 때, 반복문을 전혀 사용하지 않고 재귀 함수만을 호출하여 해당 수를 2진수로 변환해 출력하는 프로그램을 작성하세요.

###### **입력**  
첫째 줄에 10진수 정수 $N$ ($1 \le N \le 1,000,000$)이 주어진다.

###### **출력**  
변환된 2진수 값을 한 줄에 출력한다. (앞에 불필요한 0은 출력하지 않는다.)

###### **입력 예시 1**
    13

###### **출력 예시 1**
    1101

###### **입력 예시 2**
    2

###### **출력 예시 2**
    10
