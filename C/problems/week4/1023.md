# 1023. 연결 리스트 지정 위치 삽입

## **설명**  
설명 단방향 연결 리스트(Singly Linked List)의 노드를 표현하는 구조체 Node를 다음과 같이 정의합니다. 
```c
typedef struct Node {
    int data;
    struct Node* next;
} Node;
```

메인 함수에는 리스트의 시작점을 가리키는 헤드 포인터 변수(Node* head = NULL;)가 존재합니다. 이 헤드 포인터 변수의 주소와 삽입할 위치(인덱스), 그리고 데이터 값을 매개변수로 받는 삽입 함수 insertNode를 구현하세요.

함수 내부에서는 이중 포인터(Node)를 사용하여 메인 함수의 헤드 변수나 이전 노드의 next 포인터 값을 원본 수정할 수 있어야 하며, 새 노드는 반드시 동적 할당되어야 합니다.

인덱스는 0부터 시작하며, 0은 맨 앞을 의미합니다. 인덱스가 현재 리스트의 길이와 같다면 맨 뒤에 삽입됩니다. 잘못된 인덱스는 입력되지 않는다고 가정합니다.

###### **입력**  
첫째 줄에 수행할 삽입 연산의 횟수 $M$ ($1 \le M \le 100$)이 주어진다.

둘째 줄부터 $M$개의 줄에 걸쳐 삽입할 위치(인덱스)와 데이터(정수)가 공백으로 구분되어 차례대로 주어진다.

###### **출력**  
모든 삽입 연산이 끝난 후, 연결 리스트의 맨 앞 노드부터 마지막 노드까지 순서대로 데이터를 [데이터] ->  형식으로 출력하고 마지막에 NULL을 출력한다.

###### **입력 예시 1**
    4
    0 10
    1 30
    0 5
    2 20

###### **출력 예시 1**
    5 -> 10 -> 20 -> 30 -> NULL
